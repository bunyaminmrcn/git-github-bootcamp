### Section 4

```sh

# initialize the repo
git init 

# check git status
git status

# add changes to staging area from current directory and subfolders
git add .

# add a file to staging area
git add file1

# commit message
git commit -m "Commit Message"


# see logs
git log

```

```sh

# if you want to unstage changes
```
